Criando gráficos HTML5 com canvas e Charts.js

Aprenda a criar gráficos responsivos para sua aplicação web

Publicado por Leonardo Smoginski em 23 Julho de 2017

Sobre Charts.js

"Simple yet flexible JavaScript charting for designers & developers"

Charts.js é uma biblioteca em JavaScript que permite a criação de 8 tipos de gráficos diferentes, além da mescla de mais de um tipo no mesmo gráfico. Os gráficos são 100% responsivos, ou seja, se adaptam ao tamanho da janela para melhor escalabilidade. O recurso utilizado para a rendenização é o HTML5 canvas, que tem uma ótima performance em todos os navegadores modernos.

Acesse a documentação aqui.

Instalando a biblioteca

Neste tutorial nós iremos utilizar o CDN Chart.js mas você também pode baixar versão mais recente em GitHub releases.

1 - Crie um arquivo de texto e salve com extensão ".html".

<html><head></head><body></body></html>

2 - Adicione uma das URL's de importação em uma tag script.

<script src="https://cdnjs.cloudflare.com/ajax/libs/Chart.js/2.6.0/Chart.js"></script>
(Obs: só funciona com internet, para tentar offline é necessário baixar a biblioteca no GitHub.)

3 - Coloque uma tag canvas em sua página.

<canvas id="myChart" width="400" height="400"></canvas>

É recomendável atribuir um id ao seu canvas, assim fica mais fácil fazer uma referência na hora de gerar o gráfico.

4 - Inserir o JavaScript do gráfico. (exemplos)

<script>
var ctx = document.getElementById("myChart").getContext('2d');
var myChart = new Chart(ctx, {
    type: 'bar',
    data: {
        labels: ["Red", "Blue", "Yellow", "Green", "Purple", "Orange"],
        datasets: [{
            label: '# of Votes',
            data: [12, 19, 3, 5, 2, 3],
            backgroundColor: [
                'rgba(255, 99, 132, 0.2)',
                'rgba(54, 162, 235, 0.2)',
                'rgba(255, 206, 86, 0.2)',
                'rgba(75, 192, 192, 0.2)',
                'rgba(153, 102, 255, 0.2)',
                'rgba(255, 159, 64, 0.2)'
            ],
            borderColor: [
                'rgba(255,99,132,1)',
                'rgba(54, 162, 235, 1)',
                'rgba(255, 206, 86, 1)',
                'rgba(75, 192, 192, 1)',
                'rgba(153, 102, 255, 1)',
                'rgba(255, 159, 64, 1)'
            ],
            borderWidth: 1
        }]
    },
    options: {
        scales: {
            yAxes: [{
                ticks: {
                    beginAtZero:true
                }
            }]
        }
    }
});
</script>

O exemplo irá gerar um gráfico de barras, com uma cor e label diferente para cada informação inserida nos datasets.
