Iniciando Com Ionic 2

Aprenda a instalar o Ionic 2 e começar aplicações híbridas.

Publicado por Luiz Filipy em 23 Julho de 2017

Sobre Ionic 2

o IONIC, ele é um framework que usa Angular para criar aplicações com um visual muito similar ao de apps nativos. Oferecendo diversas bibliotecas de CSS e Javascript para AngularJS, ele simplifica o desenvolvimento e ajuda a produzir apps com um visual muito agradável, sem dar muito trabalho ao desenvolvedor, além de não exigir muito conhecimento prévio, apenas HTML, CSS e Javascript.

Instalando as dependências

Para criar projetos Ionic, você precisará instalar a versão mais recente da CLI e Cordova. Antes de fazer isso, você precisará de uma versão recente do Node.js. Faça o download do instalador para Node.js 6 ou superior e, em seguida, proceda à instalação do Ionic CLI e Cordova para o desenvolvimento de aplicativos nativos:

1 - Use o comando abaixo no cmd para instalar o cordova

npm install -g ionic cordova

2 - Crie o seu primeiro App Ionic.

ionic start MyIonicProject tutorial

3 - Abrindo o App.

Agora, você pode usar o comando cd dentro da pasta que foi criada.
 para poder ter um preview do app no navegador, use o comando server.
 
cd MyIonicProject/
ionic serve

Agora basta abrir o navegador e você deverá ver a mensagem "welcome to your first Ionic app
